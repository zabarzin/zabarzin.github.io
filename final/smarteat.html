SMART EAT MOBILE APPLICATION

This project is for an audience who is interested in a Mediterranean-based diet which is scientifically proven to improve Neuro health. The objective of this application is to assist a user with keeping up to date within their daily, weekly and monthly activity through this diet. And throughout the infographic design system the user is able to see their improvement and the lack of improvement within the six different food components required to be succesful with this diet.
